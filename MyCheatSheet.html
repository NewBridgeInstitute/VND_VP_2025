WHAT ARE HTML TABLES?

HTML Tables are used to display data in rows and columns. They are a great way to present data in a structured and organized way.

Tables are made up of rows and columns, and each intersection of a row and a column is called a cell. Each cell in a table can contain text, images or other HTML elements

LETS CREATE A BASIC TABLE

To create a basic table, you need to use three elements: TABLE, TR and TD. The TABLE element defines the entire table, and the TR element defines a row within the table. The TD element defines a cell within a row.
